## Português

### Sobre o Projeto
Este projeto representa um marco importante no meu início como estudante de **Análise e Desenvolvimento de Sistemas (ADS)**. Desenvolvi uma Landing Page para o aplicativo fictício Moedança, focando na aplicação prática de HTML5, CSS3 e JavaScript. 

A construção deste site foi fundamental para o meu aprendizado, pois me permitiu compreender todo o ciclo de vida de um projeto front-end: desde a estruturação do código e a criação de um layout responsivo até o uso do Git para controle de versão e a publicação do site (deploy) através do GitHub Pages.

### Integração com Inteligência Artificial
Durante o desenvolvimento, utilizei a Inteligência Artificial como uma ferramenta estratégica de suporte e consulta. A IA foi essencial para:
- **Resolução de dúvidas:** Auxiliou na compreensão de conceitos técnicos e na aplicação de boas práticas de programação.
- **Identidade Visual:** A logo do projeto foi criada com o auxílio de IA generativa.
- **Demonstração:** O GIF da seção principal foi produzido a partir de um vídeo também gerado por ferramentas de IA.

### Demonstração

## English

### About the Project
This project is a significant milestone in my journey as a **Systems Analysis and Development** student. I developed a Landing Page for a fictional app called Moedança, focusing on the practical application of HTML5, CSS3, and JavaScript.

Building this site was crucial for my growth, as it allowed me to master the full lifecycle of a front-end project: from structuring the code and creating a responsive layout to using Git for version control and deploying the site via GitHub Pages.

Throughout the process, I used Artificial Intelligence as a strategic support and reference tool. AI played a key role in:
- **Technical Support:** It helped me clarify technical concepts and implement coding best practices.
- **Visual Identity:** The project's logo was created with the help of generative AI.
- **Demonstration:** The GIF in the main section was produced from an AI-generated video.
